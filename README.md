# vscode-enable-proposed-api

VS Code extension which adds a command to restart the editor using the --enabled-proposed-api flag.

## Requirements

⚠️ Only works in macOS

## Usage

1. <kbd>⌘</kbd>+<kbd>P</kbd>
2. `> Restart with --enable-proposed-api`
